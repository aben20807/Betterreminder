## Intro
+ 電池電量、充電狀態偵測
+ 排程使用 : 可依照電量執行動作 ( 提醒、關機、睡眠、休眠 )
+ 簡易資料庫儲存動作紀錄
+ 因為某些原因所以目前只有英文版 ( 繁體中文我盡快補上QAQ
+ 可最小化至系統列 ( 會顯示目前電量 )
+ 具有 ini 檔可記錄上次設定 ( 語言、最小化的位置 )
+ 可查看所有動作執行的時間紀錄
